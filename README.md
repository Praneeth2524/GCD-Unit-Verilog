# GCD Unit using FSM-Based Controller and Datapath

## Overview

This project implements a Greatest Common Divisor (GCD) unit in Verilog HDL using a separate datapath and FSM-based controller architecture. The design computes the GCD of two input numbers using iterative subtraction.

## Features

* FSM-based controller design
* Separate datapath and control path architecture
* Modular Verilog implementation
* Comparator and subtractor-based GCD computation
* Verified using a custom Verilog testbench


## Technologies and Tools

* Verilog HDL
* Icarus Verilog
* GTKWave
